Hi, I'm Liang~

3d micropillar modeling

As part of a UROP, I'm been working on creating a software suite in Java Processing to model the geometry of high-resolution 3D micropillar structures for visualization and 3D printing. These structures are interesting because of their ability to bend and collectively represent a vector field in their direction of orientation. By inputting a prepared image, we can output a representation of the resultant hair structure based on the image properties. Unfortunately, the code isn't open source :\

Created with Jifei Ou in MIT Media Lab's Tangible Media group

impression

Visit the site here! (although it's broken right now)

My project for HackMIT 2015 - our app takes in an image and outputs a short playlist of songs recommended for you (depending on the image uploaded). Impression makes use of the Clairfai API to extract descriptor words from the image, runs an NLP script to transform those descriptors to emotion, then calls the EchoNest + iTunes APIs to create and display the playlist.

Created fall 2015 with Ajay S, Monde D, Courtney G
