tower counterfactuals

our paper trial visualizations

submitted and accepted as a talk at cogsci. the goal is to model people's counterfactual judgments of stability by generating noisy simulations of brick towers and designating a particular "special" brick to remove. we ran several amazon mturk experiments to sample human judgment and created a noisy simulation model that better represented people's intuitive judgments.

streaming normalization

source code on github

we experimented with a new normalization method in recurrent neural nets called streaming normalization, which is a generalization of familiar methods like batch and layer normalization, as well as with decoupled/delayed accumulation and update of gradients. these methods have already been published, but we have been testing further (to be revealed) extensions.

3d micropillar modeling

the conference paper

as part of a urop, i created a java processing suite to model the geometry of 3d-printed high-resolution micropillar structures. these structures have a collective ability to bend and represent a vector field in their direction of orientation. by inputting a prepared image, we can output a representation of the resultant hair structure based on image properties. it's now patented!

textura

source code on github

image feature detection, for any feature you like. given an picture and an adjective, textura will tell you how well your adjective describes your picture on a scale of 0 to 1. it automatically scrapes the web for images that match the adjective, trains a convolutional neural net based on these images, and returns a value. the model can then be cached for future use!

impression

source code on github

a project for hackmit 2015 - the app takes in an image and outputs a short playlist of songs recommended for you based on the uploaded image. impression makes use of several apis to extract descriptor words from the image, runs an nlp script to transform those descriptors to emotion, then calls some more apis to create and display the playlist.
