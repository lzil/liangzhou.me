3d micropillar modeling

the conference paper

as part of a urop, i created a software suite in java processing to model the geometry of high-resolution 3d micropillar structures for visualization and 3d printing. these structures are interesting because of their ability to bend and collectively represent a vector field in their direction of orientation. by inputting a prepared image, we can output a representation of the resultant hair structure based on the image properties. it's now patented!

spring 2015 with jifei ou in mit media lab's tangible media group

reedr

source code on github

For yhack 2015, we created a vanilla javascript app that takes in an image of text (i.e. a page from a book) and performs word segmentation (not ocr - it simply recognize blocks of words, not the characters themselves) and orders those words for potential word division. The original idea was to apply this to speed reading. we won the prize for the best hack under 4kb from datto!

fall 2015 with vahid f, anthony l

impression

link to (broken) site

a project for hackmit 2015 - the app takes in an image and outputs a short playlist of songs recommended for you (depending on the image uploaded). impression makes use of the clarifai api to extract descriptor words from the image, runs an nlp script to transform those descriptors to emotion, then calls the echonest + itunes apis to create and display the playlist.

fall 2015 with ajay s, monde d, courtney g
